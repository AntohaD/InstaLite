This application is written using react.js, using api instagram. With it, you can view the last 20 posts and go to the detailed page of the post. <br>

For starting app : in the project directory, you can run:

### `npm run dev`

For the application to work properly, you need to get your USER ID at https://www.instagram.com/developer/ and specify in the "Website URL" line: http://localhost:8080/ . <br>

The resulting USER ID must be inserted into the variable "const instaClientId", which is located at src/component/logout-login/login.jsx <br>
